Unit 10: Data Analysis

This unit covered exploratory data analysis techniques, statistical methods, and approaches for deriving insights from cleaned and integrated datasets.

Learning Outcomes

  • Apply exploratory data analysis techniques
  • Use statistical methods for data understanding
  • Create meaningful data visualisations
  • Communicate analytical findings effectively

Understanding when to use different chart types and how to avoid misleading representations was particularly valuable.
